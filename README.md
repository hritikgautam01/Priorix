## 📌 Overview

Priorix is more than a simple to-do app.

It helps users not only manage tasks but also prioritize them using the **Eisenhower Matrix**, enabling smarter decision-making and better productivity.

The application provides multiple ways to visualize the same tasks:

- 📋 Kanban Board
- 🎯 Eisenhower Matrix
- 📅 Calendar View *(Planned)*
- 📊 Productivity Analytics *(Planned)*

---

## ✨ Features

### Current

- Create, edit and delete tasks
- Multiple Kanban columns
- Drag and Drop tasks
- Task priority levels
- Due dates
- Labels & Tags
- Search tasks
- Filter tasks
- Responsive UI
- Dark / Light Mode

### Planned

- AI Daily Planner
- Authentication
- Cloud Sync

---

## 🧠 Why Priorix?

Most task management apps help users organize tasks.

Priorix helps users organize **and prioritize** tasks by combining workflow management with proven productivity principles.
